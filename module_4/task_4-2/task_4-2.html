    7. Answer to both question 6 and 7: A callback routine is used in the .sort() method in task 8 to define how the array should be sorted. A callback function, in our case function(a, b), is a function passed as an argument to another function. When sorting, function(a,b) determines the order of the element by comparing their values. If the callback return a negative number, a comes before b, or it’s a positive number b comes before a, and when 0 the order remains unchanged. In our task one use both a – b to return ascending, and b – a to return descending.
